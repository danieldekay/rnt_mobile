# PWA-How-to: installieren und aktualisieren

## Worum es geht

Die mobile RNT-Version unter [mobile.rhein-neckar-tango.de](https://mobile.rhein-neckar-tango.de) ist eine PWA. Das heisst: Du nutzt sie ueber den Browser, kannst sie dir aber wie eine App auf den Startbildschirm legen.

Der wichtige Unterschied zu einer klassischen App ist: Es gibt keinen App Store dazwischen. Installation und Updates laufen direkt ueber den Browser.

## Installation auf iPhone oder iPad

1. Oeffne [https://mobile.rhein-neckar-tango.de](https://mobile.rhein-neckar-tango.de) in Safari.
2. Tippe unten oder oben auf das Teilen-Symbol.
3. Waehle `Zum Home-Bildschirm`.
4. Bestaetige mit `Hinzufuegen`.

Danach liegt der RNT-Kalender wie eine App auf deinem Home-Bildschirm.

## Installation auf Android

1. Oeffne [https://mobile.rhein-neckar-tango.de](https://mobile.rhein-neckar-tango.de) in Chrome.
2. Falls ein Hinweis erscheint, tippe auf `App installieren`.
3. Falls kein Hinweis erscheint, oeffne das Browser-Menue und waehle `Zum Startbildschirm hinzufuegen` oder `App installieren`.
4. Bestaetige den Dialog.

Danach kannst du den Kalender direkt vom Startbildschirm aus oeffnen.

## Installation auf dem Desktop

Je nach Browser kannst du die Seite auch am Rechner installieren, zum Beispiel ueber das Installieren-Symbol in der Adressleiste von Chrome oder Edge. Fuer RNT ist das vor allem auf dem Handy relevant, technisch funktioniert es aber oft auch am Desktop.

## Wie Updates funktionieren

Die PWA wird nicht ueber einen App Store aktualisiert. Neue Versionen kommen direkt ueber die Web-App.

Die App prueft deshalb selbst, ob eine neuere Version bereitsteht. Wenn waehrend deiner Nutzung ein neues Deployment live geht, erscheint in der App ein Hinweis mit einer direkten Aktion zum Aktualisieren.

Wenn du vermutest, dass es schon eine neuere Version gibt, kannst du ausserdem unten im App-Footer auf `Jetzt auf Updates pruefen` tippen. Die App meldet dir dann, ob schon eine neuere Version bereitsteht oder ob du bereits auf dem aktuellen Stand bist.

In vielen Faellen reicht danach ein Tip auf `Jetzt aktualisieren`. Falls gerade kein Update-Hinweis sichtbar ist, reicht es meistens trotzdem:

1. die App komplett zu schliessen
2. sie erneut vom Home-Bildschirm zu oeffnen

## Wenn eine alte Version haengen bleibt

Falls du den Eindruck hast, dass noch eine alte Ansicht offen ist, gehe so vor:

1. Schliesse die installierte App komplett.
2. Oeffne sie erneut vom Home-Bildschirm.
3. Wenn die alte Ansicht bleibt, oeffne [https://mobile.rhein-neckar-tango.de](https://mobile.rhein-neckar-tango.de) einmal direkt im Browser.
4. Lade die Seite dort neu.
5. Oeffne danach wieder die installierte Version vom Home-Bildschirm.

Wenn es dann immer noch seltsam aussieht, ist der einfachste Weg:

1. die App vom Home-Bildschirm entfernen
2. die Seite noch einmal neu installieren

## Was du nicht tun musst

1. Du brauchst keinen App-Store-Download.
2. Du brauchst kein Nutzerkonto nur fuer die Installation.
3. Du musst Updates nicht manuell im Store suchen.

## Kurzfassung fuer Hilfe-Seiten oder Support

Die RNT-PWA installierst du direkt ueber den Browser. Auf iPhone/iPad in Safari ueber `Teilen` -> `Zum Home-Bildschirm`, auf Android in Chrome ueber `App installieren`. Updates kommen direkt ueber die Web-App. Die App kann selbst auf neue Versionen pruefen und zeigt dann einen direkten Aktualisieren-Button. Wenn eine alte Version haengen bleibt, die App kurz schliessen, neu oeffnen und notfalls einmal im Browser neu laden.
